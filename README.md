## To run app

1.
    ```
    conda create --name myenv

    ```

2. 
    ```
    conda activate myenv

    ```

3. 
    ```
    python -m pip install --upgrade pip

    ```

5. 
    ```
    pip install -r requirements.txt

    ```

6. 
    ```
    python app.py
    ```

Do not forget to change your .env file with valid credentials.
